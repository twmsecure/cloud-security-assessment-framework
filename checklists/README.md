# Cloud Security Assessment Checklist

This folder contains assessment checklists used to evaluate cloud security controls across public, private and hybrid cloud environments.

## Assessment Domains

- Identity & Access Management
- Network Security
- Data Protection
- Logging & Monitoring
- Vulnerability Management
- Incident Response
- Business Continuity & Disaster Recovery

## Purpose

The checklist provides a structured approach for identifying security gaps, evaluating control effectiveness and documenting remediation opportunities.
